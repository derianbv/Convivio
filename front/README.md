# Convivio - Expo React Native

Proyecto base creado para replicar las pantallas mostradas por el usuario (Home, Categorías, Áreas).

Instrucciones rápidas:

1. Instalar dependencias:

```bash
cd front
npm install
# o yarn
```

2. Ejecutar en Expo (requiere `expo-cli` o usar `npx expo`):

```bash
npx expo start
```

3. Abrir en un emulador o en un dispositivo con la app Expo Go.

Notas:
- Los assets son placeholders (imagen remota). Reemplaza por imágenes locales en `assets/`.
- La UI inicial está en `src/screens/*`.
